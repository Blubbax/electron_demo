Hello Electron!

Aufgabenstellung:

Ziel dieser Aufgabe ist die Anzeige aller E-Scooter Reservierungen in einer Tabelle. Um dies zu erreichen, sind die folgenden Schritte notwendig:

  1. Die Datei reservations.html wurde bereits angelet. Um diese Aufrufen zu können, ist diese dem Menü hinzuzufügen. Das Template für das Menü ist in der Datei index.js enthalten. Erweitere das Template um einen Menüpunkt.

  2. Erzeuge in reservations.html eine Tabelle, in der alle E-Scooter Reservierungen angezeigt werden. Als Orientierung kann die Datei scooters.html dienen.

  3. Sobald das Laden dieser Tabelle abgeschlossen ist, soll eine Nachricht an das System ausgegeben werden, in der dem Benutzer mitgeteilt wird, wie viele Reservierungen abgerufen wurden.

  4. Probiere die Funktion mit npm start aus.

  5. Exportiere diese Anwendung mit npm run make und führe sie aus. Das ausführbare Programm wird im Ordner out zu finden sein.

Voraussetzungen:

Damit dieser Client korrekt funktionieren kann, ist darauf zu achten, dass die Datenbank eigerichtet ist und die php-Skripte im htdocs Ordner liegen.
